README
Un componente esencial
de cualquier planificación del desarrollo es la información.
Sin los datos necesarios, los esfuerzos de un club para planificar el crecimiento y el bienestar
de sus integrantes no pueden asentarse en la realidad y, por lo tanto, pueden presentar graves fallos.
Es necesario definir una estragia efectiva y que todo el esfuerzo se canalice hacia el éxito!
Las investigaciones aplicadas en el deporte están estrechamente ligadas a la solución de problemas científicos de la preparación de deportistas altamente calificados. La solución de muchos de estos problemas permite hacer más eficaz el control de la preparación de los deportistas y obtener un nivel más alto de los alcances deportivos.
